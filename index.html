Featured

      
NEXA

      
AI Desktop Assistant

      
A voice-driven AI desktop assistant with 190+ callable functions, controlling your Windows environment through natural language with approximately 95% intent accuracy. NEXA runs a local GraphRAG memory system under the hood, storing personal memories as a relational node graph, visualized in an interactive graph UI, semantically retrieved via all-MiniLM and LanceDB, so every response is grounded in context that actually knows you. Fully offline-capable with online/offline LLM switching via Ollama and Llama 3.1.

Featured

      
CortX

      
Autonomous Web Research Assistant

      
CortX is a fully local autonomous web research assistant that thinks before it searches. It breaks down your query into intent, hunts sources across the web, and pushes through anti-bot barriers using a three-tier scraping engine that falls back automatically across httpx, Playwright, and Selenium. Scraped content is embedded and stored in LanceDB, powering a RAG pipeline that synthesizes clean, source-cited answers through LangChain and makes follow-up queries instant without re-scraping. Ships with a polished PyQt6 desktop GUI, full offline support via Ollama, and optional Groq cloud fallback when you need speed.
