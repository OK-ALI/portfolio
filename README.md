## 💎 Key Features

### ☄️ 1. Interactive Force-Directed Skills Neural Graph
The crown jewel of the portfolio is an interactive, force-directed skills network built from scratch using HTML5 Canvas physics:
* **Custom Starburst Physics**: Hub-and-spoke spring forces, category-to-category repulsions, and sibling collisions are simulated in real-time to create a fanned-out, floating network structure.
* **Direct Synaptic Spotlight**: Hovering over or tapping any individual skill leaf (e.g. *GraphRAG*, *Knowledge Graphs*) triggers a focused highlight. Sibling skills instantly fade out to `0.15` opacity, while a single, vibrant path lights up from the core node to the parent category and directly to the targeted skill.
* **High-Tech Signal Comets**: Continuous neon spark particles glide asynchronous along active synapse lines, utilizing Canvas shadow glows (`shadowBlur = 12`) to simulate active biological signals.
* **100% Tactile Interaction**: Supporting mouse drags, panning, and mobile-friendly tap highlights, the entire visual skill label card is fully interactive (`pointer-events: auto`).

### 💻 2. Interactive IDE Code Auto-Writer
A full-scale, sleek, syntax-highlighted IDE interface typing simulator that runs on pure JavaScript:
* **Simulated Autotyping**: Dynamically highlights Python functions, classes, and strings on-the-fly (`30-65ms` typing speed).
* **Multi-Tab Navigation**: Seamlessly toggle between different code files (`greeting.py`, `coffee.py`, `solve_error.py`) to explore my code, logic, and visual bug-squashing algorithms.

### ⚡ 3. Ultra-High Performance & Native 120fps Rendering
Highly optimized to ensure silky-smooth, lag-free scrolling across all desktop and mobile viewports:
* **Zero Layout Thrashing**: Eliminates forced reflows by caching all canvas wrapper dimensions and CSS accent variables globally. Expensive DOM offset reads and `getComputedStyle` calls are completely removed from the animation frame loop.
* **Intersection Observer Loop Pausing**: Utilizes a dedicated `IntersectionObserver` to **completely pause the physics tick loop** (`cancelAnimationFrame`) when the skills section is out of view. Resource usage drops to **exactly 0%** when scrolling other parts of the site.
* **GPU Hardware Acceleration**: Forced composite layer rendering on elements using CSS GPU triggers (`will-change: transform, opacity;` and `backface-visibility: hidden;`).

## 🚀 Local Setup & Development

Since this portfolio is built using lightweight, high-performance vanilla web standards, you can run it locally with zero dependencies:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/OK-ALI/portfolio.git
   cd portfolio
   ```

2. **Run locally**:
   * Simply double-click `index.html` to open it directly in your browser.
   * Alternatively, spin up a local development server using Python:
     ```bash
     python -m http.server 8000
     ```
     Then navigate to `http://localhost:8000` in your web browser.
